

# WheelPath AI

A React-based web application that integrates with Firebase for authentication, database management, and cloud functions. The project provides a platform for managing and interacting with shared drive authentication and related services.

## Tech Stack

- **Frontend**: React 18
- **Backend**: Firebase
  - Authentication
  - Firestore Database
  - Cloud Functions
  - Storage
- **Additional Libraries**:
  - React Router DOM for navigation
  - Axios for HTTP requests
  - React Toastify for notifications
  - React Error Boundary for error handling

## Project Structure

```
src/
├── assets/         # Static assets
├── components/     # Reusable UI components
├── config/         # Configuration files
├── contexts/       # React context providers
├── functions/      # Firebase cloud functions
├── hooks/          # Custom React hooks
├── pages/          # Page components
├── platforms/      # Platform-specific code
├── services/       # Service layer for API calls
├── styles/         # Global styles
└── utils/          # Utility functions
```
